Tags and Attributes

Tags

Tags surround the content of your document and give meaning to it, there are different type of tags like Block-level tags and Inline tags which can be nested inside block level tags. They usually come in pairs with begin and end tag for example <h1> ... </h1>, other tags are self-closing tags like <br /> and <hr /> tags

Block-level tags

Block-level tags provide a basic content structure for a web page. When block-level elements are displayed in a browser usually they start and end with a new line. Block-level tags we are going to cover are:

  • Headings
  • Paragraphs
  • Lists

Headings are one of the most important structure components in an HTML document. It allows the content of a web page to be divided in different sections, therefore the reader of a web page does not need to read all the content of a page to find what he is looking for, but only the relative section. There are six levels of headings <h1> (which is the most important) to <h6> (which is the least important). There must:

  • Be only one <h1> per page
  • They cannot be nested together
  • Levels should not be skipped.

More information about headings can be foundhere.

Paragraphs of an HTML document have the same use as the paragraphs of a text document. However, in an HTML document, paragraphs must be define with <p> and </p> tags.

Web browsers will automatically add a new line before and after a paragraph. However, if you want to insert a new line without starting a new paragraph, you can use the <br /> tag, you can also insert a new line inside the <p> tags but you should avoid doing it. Additionally you can divide the HTML document in different sections using the <hr /> to insert a new horizontal line. 

<!DOCTYPE html>
<html>
<head>
<title> A Simple (HTML 5) Document </title>
<meta charset="utf-8"/>
</head>
<body>
<h1> A Simple document</h1>
<p> A paragraph.</p>
<hr/>
<p> A second <br/>paragraph.</p>
</body>
</html>

Click the button to see the above HTML document in a web browser

There are three types of lists to use in an HTML document. The ordered list, the unordered list and the description list which is used when describing a term or a name.

Unordered List

<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>

Ordered List

<ol>
<li>First item</li>
<li>Second item</li>
</ol>

Description List

<dl>
<dt>Item</dt>
<dd>Something you can use</dd>
</dl>

Inline Tags

Inline tags must be nested inside block-level tags. There are two types of inline tags, the logical inline tags which provide additional semantic meaning to the content of an HTML document, and the physical inline tags which are mainly used to produce a visual effect on an HTML document.

Tag Enclosed content is: Visual feedback
<em> given extra emphasis Text in italics
<strong> given strong emphasis Text in bold
<cite> a citation or reference Usually italicised
<code> is program code Text in monospace font
<abbr> and <acronym> is a shortened form None - requires a title attribute

Attributes

Attributes provide additional information about the elements of an HTML document. They always come in name/value pairs (name=”value”). Attributes are declared in the opening tag, self-closing tags can contain attributes as well. The attributes below can be applied to any <body> tag.

Attribute Purpose
class Associates an element with a CSS class
id Uniquely identifies element for CSS/Scripting
style Provides inline CSS style rules for an element
title Describes an element and its content. Creates tool-tips in browsers and used by screen readers.
