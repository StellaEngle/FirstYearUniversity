Create a Simple Web Page

To create a simple document in HTML 5 you have to include:

  1. A line containing HTML version information
  2. A declarative header section (delimited by the <head> element)
  3. A body which contains the document’s actual content. The body must be implanted by the <body> element

Sections 2 and 3 should be delimited by the <html> element.


Copy and paste the text below in a text editor and save it as html document (.html) and then open in a browser.

<!DOCTYPE html>
<html>
<head>
<title> A Simple (HTML 5) Document </title>
<meta charset=”utf-8”/>
</head>
<body>
<h1> A Simple document</h1>
<p> A paragraph.</p>
</body>
</html>
