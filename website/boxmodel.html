CSS Box Model

Every element on a web page can be represented as a box. Each box has some properties.

  • Border
  • Padding
  • Margin

The border of a box is made up from three other sub-properties:

  • Width ( border-width: 10px; )
  • Color ( border-color: red; )
  • Style ( border-style: solid; )

Padding creates space inside the box ( padding: 10px; ) and margin creates space between boxes ( margin:10px; ).
