Images

To add an image to a web page you need the self-closing tag <img /> and some attributes inside of it. The attributes you can use are the “src” it specifies a URL location from which an image can file be found. The “alt” attribute is a small description of the image to be used by the readers of a web page, or if the image file fails to load and is required to be used. The “width” and “height” attributes define the dimensions in which the browser will display the image file, however they do not change the size or the shape of the image. Lastly, the “title” attribute is a descriptive tag of the image, when you mouse over the image you can see the value of the title if it exists.

<!DOCTYPE html>
<html>
<head>
<title> A Simple (HTML 5) Document </title>
<meta charset="utf-8"/>
</head>
<body>
<h1> A Simple document</h1>
<div>
	<img src="img3.jpg" alt="Image" title="This is an image">
</div>
<p> A paragraph.</p>
</body>
</html>

Multimedia Objects

You can also add to an HTML document and some other media objects as well, expect from images. For example audio content (e.g. streamed music), video content (e.g. movies and videos from YouTube) or other embedded applications (e.g. flash application).

To add a media object you can use tags <object> or <param>. Moreover HTML 5 supports new logical tags, which you can use, like <video>, <audio>, <iframe> and <embed>.
