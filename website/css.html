CSS

There are two different kinds of style sheets, the internal style sheets and the external style sheets. To use an internal style sheet you need to use the <style> tags inside the <head> section of an HTML document. To use an external style sheet, you have to link it to an HTML document. You can also have multiple external style sheets linked to one document.

Internal Style Sheet

<!DOCTYPE html>
<html>
<head>
<title> A Simple (HTML 5) Document </title>
<style type="text/css">
body{
	background-color: grey;
	}
</style>
<meta charset=”utf-8”/>
</head>
<body>
<h1> A Simple document</h1>
<p> A paragraph.</p>
</body>
</html>

External Style Sheet

<!DOCTYPE html>
<html>
<head>
<title> A Simple (HTML 5) Document </title>
<link rel="stylesheet" type="text/css" href="mydesign.css" />
<meta charset="utf-8"/>
</head>
<body>
<h1> A Simple document</h1>
<p> A paragraph.</p>
</body>
</html>

CSS Selectors

Selectors are used to find a particular element in a HTML document and then format it using the CSS file. They can be combined together to give more control over the elements of the document, that you wish to format.

There are three selector types. The tag selector which matches all instances of a tag, e.g. </p>, <h1>. The class selector which matches all the tags which contain the particular class attribute. Last but not least, is the ID selector, which matches the unique tag which contains the particular ID attribute.

The table below shows some of the Selector Syntax you can have.

Selector Pattern matched
p All <p>
.items <anytag class="items">
p.items All <p class="items">
#part1 The only <anytag id="part1">
#part1 p All <p> nested anywhere inside the only <anytag id="part1">
#part1 > p All <p> that are direct children of <anytag id="part1">
#part1 p.items All <p class="items> nested anywhere inside the only tag with the id of part1
div#part1 p All <p> nested anywhere inside the only <div id="part1">

CSS Units

CSS supports many types of measurement unit. Absolute units are calculated independently of the other content of a web page and browser defaults. Absolute units include pixels (px), points (pt), and millimetres (mm). Relative units are calculated proportionally against other content of a web page. Relative units include percentages (%), eme (em), exes (ex) and special relative units for text such as small, large etc.
